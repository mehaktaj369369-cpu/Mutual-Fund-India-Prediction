Mutual Funds India Prediction

## Project Overview

The Mutual Funds India Prediction project is a Machine Learning-based system designed to predict mutual fund performance using historical financial data.
The model analyzes key financial indicators such as Assets Under Management (AUM), expense ratio, risk level, and past returns to generate performance predictions.

## Objective

* Predict mutual fund returns using machine learning algorithms.
* Analyze important financial features affecting fund performance.
* Compare multiple models to identify the best-performing algorithm.

## Project Structure

* Project Report – Detailed documentation of the project.
* Synopsis – Summary of the project objectives and methodology.
* Presentation (PPT) – Visual explanation of the project.
* Dataset – Raw and cleaned financial data.
* Notebook / Source Code – Python implementation with ML models.


## Dataset Features

| Feature       | Description                         |
| ------------- | ----------------------------------- |
| AUM           | Assets Under Management             |
| Expense Ratio | Fund management charges             |
| Risk Level    | Risk classification of the fund     |
| 1-Year Return | Target variable (Prediction Output) |
| 3-Year Return | Historical performance              |
| 5-Year Return | Long-term performance               |
